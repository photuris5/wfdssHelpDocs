New Analysis

This tab is where you create/request an analysis, run an analysis, and review or modify information on analysis results

Create an Analysis

NOTE: Creating an Analysis is only available to Fire Behavior Analysts, Geographic Area Editors, and Support Roles.

  1. Navigate and open the incident.

  2. Select the Analysis stepper and the New Analysis tab

    1. Choose your Analysis Type (FARSITE, FlamMap, FSPro, MTT), select Create Analysis.

  3. Enter the Analysis Name.

  4. Enter the Analysis Start Date.

  5. Click SAVE ANALYSIS to save the name and start date or click CLOSE to return to the .New Analysis tab without saving any information.

Requesting an Analysis

NOTE: Requesting an Analysis is available to all non- Fire Behavior Analysts, non-Geographic Area Editors, and non-Support Roles

  1. Navigate and open to your incident.

  2. Select the Analysis stepper and the New Analysis tab

  3. Choose your Analysis Type <Add links (Farsite, FlmaMap, FSPro, or MTT)>>, select Request Analysis.

  4. Enter the Analysis Name

  5. Enter the Analysis Start Date.

  6. Enter a note explaining the request tothe Analyst that accepts it.

  7. Click .

  8. Click ACCEPT ANALYSIS to begin working the analysis.

Enter Analysis Information

  1. Expand the General accordion. Review and/or modify the Analysis name, Analysis Start Date, Number of Fires, Number of Days, Crown Fire Method (accept Finney (1998) or select Scott/Reinhardt (2001))

  2. Enter the Ignition File name. When you are missing an Ignition, create one using the Drawing Tool on the map and name the ignition File.

  3. Select the Barrier File. When you are missing a Barrier File, create one using the Drawing Tool on the map and name the Barrier File. Ensure the Barrier File does not overlap the Ignition File.

  4. Deselect the Barrier File checkbox as appropriate.

  5. Click SAVE.

  6. Expand the Notes accordion.

  7. Select the type of note from the drop-down menu: General Input, Weather, Landscape, Results, or Status.

  8. Enter the note text which may not exceed XXXXX characters.

  9. Click ADD to save the note text or DELETE to remove the note text. 

  10. Expand the Existing Notes accordion.

  11. Select the note you want to edit. Click SAVE when finished.

  12. Click IMPORT NOTES to import them to the decision you are working on, DELETE ALL notes, or CLOSE to return to the NEW ANALYSIS tab without saving any information.

Enter Landscape Information

  1. Select the Landscape tab. If you have not previously created a landscape, you can do so here using the Drawing Tool on the map.

  2. Expand Edit Landscape and enter one or more landscape rules by clicking ADD TO RULES for each rule you enter. You can clear landscape rules by clicking DELETE.

  3. Expand the Landscape Details accordion. 

  4. Enter the Extent Name and Landscape Details.

  5. Enter landscape information: North, South, East, West latitudes, Resolution, Fuel Model, and Data Source.

  6. Click CREATE LANDSCAPE.

  7. Expand the Notes accordion.

  8. Select the type of note from the drop-down menu: General Input, Weather, Landscape, Results, or Status.

  9. Enter the note text which may not exceed XXXXX characters.

  10. Click ADD to save the note text or DELETE to remove the note text. 

  11. Expand the Existing Notes accordion.

  12. Select the note you want to edit. Click SAVE when finished editing. 

  13. Click IMPORT NOTES to import them to the decision you are working on, DELETE ALL notes, or CLOSE to return to the NEW ANALYSIS tab without saving any information.

  14. Click CLOSE to return to the New Analysis tab without saving any information as appropriate.

Enter Weather Information

  1. Expand the ERC Classes accordion.

  2. Select the Weather Source from the drop-down menu.

  3. Select the Station from the drop-down menu. Verify the Station Details.

  4. Select the Date Filter from the Start Year and End Year from the drop-down menus.

  5. Select the Date Filter from the Start/Month Date and Stop/Month Date from the drop-down menus.

  6. Select the Fuel Model from the drop-down menu.

  7. Click GENERATE ERC CLASSES. Runs the ERC Classes and displays them.

  8. Click the appropriate checkbox(es) to select the row. You can click ADD ROW to add a row or DELETE SELECTED to delete a row.

  9. Click GENERATE ERC STREAM. Displays the ERC data. Modify the ERC Stream data if needed.

  10. Select the number of days to run weather information from the drop-down menu.

  11. Select the ERC Type from the drop-down menu.

  12. Select the Winds Type from the drop-down menu.

  13. Click GET FORECAST. Displays the forecast information.

  14. Click the appropriate checkbox(es) to select the row. You can click ADD ROW to add a row or DELETE SELECTED to delete a row.

Enter Winds Information

  1. Expand the Winds accordion.

  2. Select the Weather Source from the drop-down menu.

  3. Select the Station from the drop-down menu. Displays the station details.

  4. Select the Date Filter from the Start Year and End Year from the drop-down menu.

  5. Select the Start Month/Day and Stop Month/Day year.

  6. Select the Start Hour and End Hour from the drop-down menus.

  7. Select the Winds Type from the drop-down menu.

  8. Click GENERATE WIND MATRIX. Displays the wind matrix information.

  9. Click the appropriate checkbox(es) to select the row. You can click ADD ROW to add a row, or DELETE SELECTED to delete a row.

  10. Expand the Weather Notes accordion.

  11. Select the type of note from the drop-down menu: General Input, Weather, Landscape, Results, or Status.

  12. Enter the note text which may not exceed XXXXX characters.

  13. Click ADD to save the note text or DELETE to remove the note text. 

Existing Notes

  1. Expand the Existing Notes accordion.

  2. Select the note you want to edit. Click SAVE when finished editing. 

  3. Click IMPORT NOTES to import them to the decision you are working on, DELETE ALL notes, or CLOSE to return to the NEW ANALYSIS tab without saving any information.

  4. Click SAVE to save the information or click CLOSE to return to the New Analysis tab without saving any information.

Run the Analysis and View Results

  1. Expand the Run accordion. Displays a list of required items. The items completed are checked off.

  2. Click SUBMIT ANALYSIS.

  3. Expand the Outputs accordion. Displays the analysis outputs.

  4. Expand the Statusing accordion. Displays status information.

  5. Click SAVE. his sets the status value of the analysis to "review", Once you do this, you cannot return this analysis to any status before "review" on the "Analysis Status.
