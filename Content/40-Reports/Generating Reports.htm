Generating Reports

The Reports tab is currently active for WFDSS Classic Legacy Data prior to 03/26/2025. PDFs are available for published decisions. Currently, the other reports features are under development.

Incidents can be searched for by:

  • IRWIN ID

  • Incident Name

  • Incident Size

  • Estimated Final Coast

Additional search filters are:

  • Discovery Date

  • Any Out Date, Out Date, or No Out Date

  • Incident has Published Decision

  • Incident has Completed Analysis

  • Geographic Area

  • State

  • Jurisdictional Agency

  • Jurisdictional Unit

  • Protecting Unit

Data Download

When an incident is located, click on the name of the incident to initiate download of:

  • Incident Information

  • Decisions

  • Analysis Data

  • Other Incident Information

Click the download choice to generate Incident Information PDF, Decisions zip file of PDFs, Analysis Data zip file or Other Incident Information zip file.

The zip files take a few moments to generate. There will be a download percent complete on the screen.

Decision Content

The decisions download provides a zip file that contains a PDF of each published decision for an incident.

Analysis Content

The analysis data zip contains individual folders for each model type.

In each model type folder, the runs will be located in individual folders. Folders are labeled with the model run number.

 

In each of the model runs, there will be an analysis report labeled with the analysis name. Folders for inputs and outputs for the analysis are also included with each analysis folder.

The inputs folder contains the file used by the model software to run. The landscape critique and all shapefiles used in the model are included.

The outputs folder contains all the files that used to be available for legacy incident analysis downloads.

Other Incident Info Content

Incident Info Content contains folders of all the images created or uploaded for published decisions. It also contains reports related to the decision like the Flame Act report or any other custom report created to support an incident. The Incident History file contains a spreadsheet of all actions completed for an incident decision.

Image content available for download:

  • Maps

  • Miscellaneous

  • Weather

Incident History Spreadsheet contains:

  • Date

  • User

  • Action

  • Comments

  • Relative Risk Rating

  • Org Assessment
