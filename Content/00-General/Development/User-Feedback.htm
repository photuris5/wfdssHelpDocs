User Feedback

 

We want your feedback... 

If you would like to review previous user group sessions, they can be found on the WFM RD&A YouTube channel Its important to note that these sessions were recorded some time ago and the application has changed to varying degrees depending on feature area since recording. Additionally, although the feedback forms are closed if you have feedback to add you can check to see if it's already been captured in the feedback inventory, which is updated periodically, if your feedback hasn’t been captured you can submit it by filling out the needs and wants form. If you would like to actively participate in User Group feedback gathering you can apply at this form.
