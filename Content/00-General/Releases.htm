WFDSS v1.0.1 Release Notes (05/15/2025)

Fixes

508 Compliance & UI Standards

• Fixed an issue causing button labels on the User Search Menu to be cutoff at certain screen resolutions.

Analysis

• Fixed a bug in which the status of an analysis could be saved without the required fields completed.

• Updated the Data Source options on Landscape Extent; they are listed as most recent first for full refresh versions, followed by REMAP, and then MOD-FIS layers.

• Fixed an issue causing Incident Collaborators to receive a red error message and prevents them from viewing the Analysis summary.

• Fixed an issue that prevented the downloading of some FSPro outputs if only partial results are available; users can now download the partial results.

• Fixed an issue preventing Landscapes from creating if they included an edit with a mask.

Chart and Image Integration

• Addressed lower environment issue relating to graphs not displaying in correct location in Relative Risk.

• Fixed an issue to prevent the Fire Area/Collaborators from section from being available if the user had not initiated a decision on a new incident OR clicked on the decision button on an incident that has an decision started already. Fixed an issue preventing the Pie Charts in the Fire Area from displaying until the Spatial Inventory was generated.

Cleanup

• Old URL for homepage updated in Production.

• The User List sorting was updated to enforce alphabetical order regardless of capitalization of names.

Decision Approval/Assessment

• Fixed an issue that retrieved excessive and unnecessary data when users navigated to incidents.

• Fixed a bug relating to Decision Approver Actions; Decision Approver fields can only be saved by Approvers if they are denoted as Approver(s) in "Incident Collaborators" section or if they have GAE or Support Roles.

Environment Setup

• Fixed the build process for resource proxy, establishing reliable and consistent method for creating and deploying proxy servers that access and distribute resources.

IRWIN Writing

• Fixed a bug in which an uploaded incident perimeter did not correctly write to the IRWIN system for calculated acres.

Map Basics

• Fixed an issue preventing multi-part point shapes from uploading.

• Fixed an issue causing uploaded zip files to fail if the file extension was capitalized.

• Fixed an issue causing shapefiles that contained "M" and "Z" values to fail.

• Fixed an issue that allowed new incident points as well as drawn and uploaded perimeters to have a currency date in the future; these shapes cannot have future dates.

Map Performance

• Fixed an issue preventing Landscape Images from displaying at zoom levels of 1:577,791 and more.

• Fixed an issue in which the Current Planning area did not display on the map when using the Print Map feature.

PDF Generator

• Fixed an issue that prevented the Decision PDF from being accessible if the name of the incident had a "/" in the name.

• Fixed an issue that was preventing proper formatting on the Estimated Cost in the Decision PDF.

Strategic Course of Action

• Fixed an issue preventing users from uploading User Defined Strategic Course of Action shapes.

UI Basics

• Fixed an issue preventing the My Incident Watchlist from reflecting the most current Lead Collab/Point of Contact.

• Addressed lower environment issue relating to editing the collaborator list.

Features/Improvements

508 Compliance & UI Standards

• Tested Relative Risk section for 508 compliance, adjustments for conformity will be addressed in future releases.

• Improved 508 compliance with sections in the Relative Risk including screen reader features.

• Improved 508 Compliance for screen readers for some buttons and toggles.

Analysis

• Ensured user receives clear message on screen to indicate when weather data is in the process of saving and when weather data is saved.

• Ensured the historical daily ERC values in the FSPro Input file are ordered chronologically to support proper model function.

• Removed the current (incomplete) year from the historic Energy Release Component (ERC) calculations for ERC Classes in FSPro.

• Improved the data call to the Fire Environment Mapping System (FEMS) for Energy Release Component (ERC) data.

Chart and Impact Integration

• Updated the title of the pie chart for the Suppression Difficulty Index to reflect is it for the 97th percentile.

Cleanup

• Ensured the Approver Checkbox selection is retained when Saved on the User Profile.

• In the Weather section of the Fire Area replaced "Weather details go here" with "This Area Under Development" to improve user understanding.

• Re-ordered the Data Source options from the Landfire Product Service to ensure the latest Non- ModFIS layer is the default and presents in most recent to least recent order. Landfire 2014 was removed as an option.

• Changed verbiage in User Preferences and Profiles to improve user understanding.

• Improved Incident Details section: corrected wording, formats, selections, and removing placeholder text.

• Removed excess tabs from User Profile section and added placeholder text in the User Notification section.

• On the My Home tab, removed My Activity Section; In the My Active Incidents section, improved character spacing, column names, column order.

Decision Approval Assessment

• The Relative Risk Assessment section is now Read Only after an incident has an out date.

• Adjusted the Decision Approval & Assessment Accordion to enforce proper access and use based on user role and “Ready for Approval” button is selected.

• The Cost Estimation Narrative now provides a Rich Text Editor and is a required field.

Environment Setup

• Fixed/updated memory allocation for node build.

• Updated the ArcGIS Server URL for Analysis in all environments.

Legacy Data

• Built the User Interface for future Legacy Data searching. Note this is the Interface only, the functionality is forthcoming.

Map Basics

• Added validation to the dates for Perimeters and New Incidents to ensure backend connections to services aren't broken.

• Improved error reporting when shape uploads fail.

Map Layers

• Updated the Strategy option for "Confine/Contain" to "Confine."

• Fixed broken URL link for NWS spot weather forecasts.

• Updated Spatial Fire Planning Layer names to reflect new names/conventions.

PDF Generator

• PDF Decision links contain the full incident name.

• The decision PDF supports the Rich Text Editor in the Cost section.

Strategic Course of Action

• Changed the verbiage in the Strategic Course of Action to be more intuitive.

UI Basics

• Created deployment guide to document production build and deploy for EDU, OAT and PRD environments.

• Improvements were made to the Editor/Collaborator feature including incidents with decisions reflecting the correct collaborators and POC and ability to edit them.

• Removed the button for bulleted lists from text editor menu in decision content as it breaks the publishing of the decision PDF.

• Added a cancel button to a pop-up in case a user accidentally un-toggles "Ready For Publication”.

• Updated the Banner color for each environment; this aids users to differentiate the environments. Production is Charcoal, Education is Light Green, UAT is Black, QA is Dark Green, Dev is Purple.

User Roles

• Updated to prevent non-collaborator Editors from being able to edit the Relative Risk Assessment.

• Updated to prevent non collaborator Editors from being able to edit information on the Incident Info tab.

WFDSS v1.0 Release Notes (03/26/2025)

The WFDSS NextGen version 1.0 application has been released and includes minimum viable product for searching for an incident, starting a decision, creating an FSPro Analysis, entering decision information and creating a decision document.  The major feature areas included in this release are as follows:

  • FamAuth roles and Authentication login integration
  • My Home
  • Support Dashboard
  • User Support
  • User Profile
  • User Search
  • Incident Browsing
  • Incident Info
  • Incident Decision
  • Interactive map with outside map layers
  • WFDSS Initiated Incident Points
  • Map Tools
  • Map Layer List
  • Print Map
  • Planning Area
  • Ignition Shapes
  • Mask Shapes
  • Shape Downloads
  • Shapefile Upload Tool
  • Spatial Inventory
  • Landscape Extent
  • Fire Perimeter
  • Barrier Shapes
  • Incident and Analysis shapes saved to NIFC
  • Ck 5 Rich Text Editor
  • Analysis Summary
  • Create/Request FSPRO Analysis
  • LANDFIRE landscape submission
  • Landscape rules
  • FSPRO Results downloads
  • Analysis Notes
  • Strategic Course of Action
  • Strategic Assessment
  • Cost
  • Relative Risk
  • Organizational Assessment
  • Decision Rationale
  • Publishing Decision
