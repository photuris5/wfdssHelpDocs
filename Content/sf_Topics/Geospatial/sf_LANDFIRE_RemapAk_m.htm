LANDFIRE Data Products - Alaska

All LANDFIRE Data Product features for Alaska reflect LANDFIRE 2023 updates .

  • Canadian Forest Fire Danger Rating System (CFFDRS), which reflect LANDFIRE 2020 updates .

  • Fuels Characterization Classification System (FCCS), which reflect LANDFIRE 2020 updates .

Identify Widget and LANDFIRE Data Products

LANDFIRE Data Products will return a value, rather than a name, when using the Identify widget, similar to the image below. This value may be crosswalked to a name using one of the applicable csv spreadsheets linked at the bottom of this topic. For Example, looking up the EVT value of 7016 in the EVT spreadsheet, shows this EVT to be "Colorado Plateau Pinyon-Juniper Woodland"

LANDFIRE Spreadsheets

Click on a link below to download the appropriate spreadsheet:

  • Canadian Forest Fire Danger Rating System (CFFDRS) *

  • Forest Canopy Base Height (CBD)

  • Forest Canopy Base Height (CBH)

  • Forest Canopy Cover (CC)

  • Forest Canopy Height (CH)

  • Existing Vegetation Height (EVH)

  • Existing Vegetation Type (EVT)

  • Fire Behavior Fuel Model 13 (FBFM13)

  • Fire Behavior Fuel Model 40 (FBFM40)

  • Fuel Characterization Classification System (FCCS)

  • Vegetation Condition Class (VCC)

  • Vegetation Departure Index (VDEP)

* CFFDRS is only available for Alaska.
